# AI Skill Intelligence Platform

An AI-powered career intelligence system designed to assess student skills, identify gaps, and provide personalized learning paths along with internship and job recommendations.

##  Overview

This platform enables students to:
- Take adaptive skill assessments
- Analyze strengths and weaknesses
- Receive AI-driven learning recommendations
- Explore internship and job opportunities

Built with a scalable full-stack architecture using modern web technologies.

---

##  Tech Stack

Frontend:
- React.js
- HTML5, CSS3
- JavaScript

Backend:
- Django
- REST APIs
- OAuth Authentication

Database:
- PostgreSQL / SQLite (based on environment)

---

##  Key Features

- Role-based dashboards (Student / Admin)
- Secure OAuth login system
- Adaptive skill testing engine
- AI-driven recommendation logic
- Internship & job matching module
- Scalable API-based backend

---

##  Architecture

- RESTful backend API

- Modular frontend components
- Secure data handling practices
